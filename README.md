# بوت جمهورية السادس 🎓

بوت تيليغرام تعليمي لطلاب السادس الإعدادي مع اشتراك إجباري وإدارة محتوى كاملة.

---

## خطوات التشغيل

### 1. إنشاء البوت
- افتح تيليغرام وكلم [@BotFather](https://t.me/BotFather)
- أرسل `/newbot` واتبع التعليمات
- انسخ الـ **Token**

### 2. إعداد config.py
افتح ملف `config.py` وعدّل هذه القيم:

```python
BOT_TOKEN = "ضع_توكن_البوت_هنا"       # التوكن من BotFather
CHANNEL_USERNAME = "جمهورية_السادس"   # يوزرنيم القناة بدون @
ADMIN_IDS = [123456789]               # آيديك على تيليغرام
```

> لمعرفة آيديك، كلم [@userinfobot](https://t.me/userinfobot)

### 3. تفعيل Force Subscribe
- اجعل البوت **أدمن** في القناة بصلاحية "إضافة أعضاء"

### 4. تثبيت المكتبات وتشغيل البوت

```bash
pip install -r requirements.txt
python bot.py
```

## هيكل الملفات

```
bot-t/
├── bot.py              ← الملف الرئيسي
├── config.py           ← الإعدادات
├── database.py         ← قاعدة البيانات
├── requirements.txt
├── handlers/
│   ├── start.py        ← /start والاشتراك الإجباري
│   ├── sections.py     ← عرض الأقسام والمحتوى
│   └── admin.py        ← لوحة التحكم
├── utils/
│   ├── keyboards.py    ← الأزرار
│   └── force_subscribe.py ← التحقق من الاشتراك
└── data/
    └── bot.db          ← قاعدة البيانات (تُنشأ تلقائياً)
```

## أوامر الأدمن

| الأمر / الزر | الوظيفة |
|---|---|
| `/admin` | فتح لوحة التحكم |
| `➕ إضافة محتوى` | إضافة ملف/نص لأي قسم |
| `🗑 حذف محتوى` | حذف عنصر من أي قسم |
| `📊 إحصائيات` | عدد المستخدمين والمحتوى |
| `📢 إرسال رسالة جماعية` | إرسال لجميع المستخدمين |
